# Introduction

## 什么是 Reviewsup.io？

[**Reviewsup.io**](https://reviewsup.io) 是一个开源的评价与推荐词管理平台，帮助你**快速集成用户评价模块到网站中**，同时便捷地收集和管理来自各个渠道的用户反馈。

你可以通过导入第三方评价（如 Trustpilot、G2、Product Hunt、Twitter），或手动添加内容，生成丰富多样的组件（Widgets），仅需几分钟即可将专业的评价模块嵌入到任何网站中。

此外，Reviewsup.io 提供完善的 API 接口，支持通过邮件自动收集用户推荐词，大幅降低开发与运营成本。

## ✨ 核心功能

* **多渠道导入**：自动导入来自 Trustpilot、G2、ProductHunt、Twitter 等平台的用户评价，也支持手动添加。
* **组件嵌入**：生成支持 React、Vue 或纯 JS 的 Widgets，样式可自定义，几分钟完成集成。
* **推荐词收集**：创建自定义表单，支持通过 API 或邮件方式向用户收集推荐词并自动入库。
* **灵活管理**：统一后台管理所有评价内容，支持筛选、排序、批量发布/隐藏等操作。

* MIT 协议，完全免费，支持 Cloud-hosted 与 Self-hosted 两种部署方式。

### ⚡ 极速集成，开发成本低

* 丰富的内置组件和交互效果，覆盖主流设计样式，减少重复开发。
* 提供清晰的 API 文档与 SDK，开发者可快速完成邮件系统或产品集成。

* 内置数十种高质量样式，适配不同 UI 风格的网站。
