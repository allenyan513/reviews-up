# Reviews

reviews是reviewsup.io的一个关键功能，我们可以通过手动填写或者导入第三方reviews来丰富我们的reviews库。并且提供了查看，编辑，删除，隐藏/显示状态，置顶，精选，排序，过滤等功能来管理reviews。

## 2. 手动添加 Reviews

手动添加reviews非常简单, 通常是只需要填写一些关键信息即可,通常只有full name和message是必须的。
一个reviews的基本信息包括：
1. full name (必须)
2. email
3. title
4. profile link
5. avatar
6. message (必须)
7. upload image
8. upload video

## 3. 导入第三方平台的reviews

点击右上角的 Import Reviews 按钮，选择您想要导入的reviews来源。Reviewsup支持从多个平台导入reviews, 目前支持和即将支持的

1. x/twitter ✅
2. Trustpilot 🚧
3. Product Hunt 🚧
4. Google 🚧
5. G2 🚧
6. Facebook 🚧
7. Instagram 🚧
8. LinkedIn 🚧


### 2.1 导入 Twitter Reviews

reviewsup.io 提供了一个简单的方式来导入 Twitter 上的 reviews。只需要填写x/twitter链接， 我们会自动抓取twitter上的内容，比如poster, 文字，图片和视频等。
