# Introduction

## What is Reviewsup.io?

**[Reviewsup.io](https://reviewsup.io)** 是一个开源的，评价和推荐词管理系统。通过导入第三方或者手动创建评价和推荐词，创建widgets，可以在几分钟内将评价模块集成到你的网站中。丰富的内置widget组件样式和交互，让你的网站看起来更专业。同时通过调用我们的API接口，可以快速集成邮件发送问卷和收集用户的评价和推荐词，节约了开发时间和成本。


## 目标用户 who is this for
1. 网站开发者和设计师：需要快速集成评价模块到网站中，提升网站的专业性和可信度。
2. 企业和品牌：希望收集用户的反馈和推荐词，以便改进产品和服务，同时展示用户的满意度。
3. 营销人员：需要通过用户评价和推荐词来提升品牌形象和市场营销效果。
4. 开源项目维护者：希望通过收集用户的评价和推荐词来提升项目的可信度和吸引力。
5. 任何需要管理和展示用户评价和推荐词的个人或组织。

## 核心问题和解决方案 problem and solution

**问题1**：如何在**几分钟**内将第三方平台上的评价内容集成到自己的网站中？
- **解决方案**：提供多种widget组件样式和交互，支持React组件、Vue组件和JS代码片段的嵌入。

**问题2**：如何在**几分钟内**为现有系统增加一个评价和推荐词的收集模块?
- **解决方案**：reviewsup.io提供API接口, 开发者和网站运营人员，可以在合适的产品生命周期调用api来给用户发送收集问卷邮件，用户填写完成后，数据自动收集到系统中,然后进行管理。

**问题3**：产品冷启动阶段，如何通过评价和推荐词来为产品增加可信度？
- **解决方案**：可以通过我们的campaign功能，设计冷启动邮件，主动发送给内测用户，收集他们的评价和推荐词，并展示在网站上。

## 主要功能 key features
它的几个主要功能包括：

- 可以自动从第三方平台（如trustpilot,g2,producthunt,twitter)导入评价或者推荐词,也可手动添加
- 可以创建widgets, 编辑widgets的布局和样式，嵌入到你的网站中，支持 React组件,Vue组件 和JS代码片段的嵌入
- 可以创建和编辑收集表单，通过API调用或者邮件的方式收集用户的评价和推荐词
